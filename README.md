# 转录组数据综合分析系统 V1.0

## 项目简介

转录组数据综合分析系统是一套基于 R Shiny 开发的生物信息学数据分析软件，面向 GEO 表达矩阵整理、多数据集合并、差异分析、共表达网络分析、富集分析、机器学习建模、ROC 验证、列线图建模和免疫浸润分析等常见转录组研究流程。

软件采用模块化结构设计，各分析功能以独立模块形式组织，用户可通过图形界面上传表达矩阵、样本分组文件或基因列表，完成数据预处理、统计分析、可视化展示和结果文件导出。项目适合用于科研数据初筛、候选基因分析、图表生成和软件著作权材料整理。

## 功能模块

本项目的功能模块以 `app.R` 中注册的模块为准，目前包括以下内容：

1. **GEO 数据整理**
   - 支持上传 GSE 表达矩阵和 GPL 平台注释文件。
   - 可将探针表达矩阵整理为以 `geneSymbol` 为第一列的表达矩阵。
   - 输出结果可作为后续分析模块的统一输入文件。

2. **数据集合并**
   - 支持多个表达矩阵文件合并。
   - 根据不同数据来源或批次信息进行批次效应校正。
   - 输出校正后的表达矩阵 `geneMatrix.txt` 和 `geneMatrix.csv`。
   - 提供箱线图、PCA 图和组合图用于查看校正前后效果。

3. **差异分析**
   - 基于表达矩阵、对照组样本列表和实验组样本列表进行差异基因筛选。
   - 支持 logFC、P 值类型和阈值参数设置。
   - 输出差异分析结果表、上调/下调基因列表、火山图、热图和 PCA 图。

4. **WGCNA 分析**
   - 构建基因共表达网络。
   - 识别表达模式相似的基因模块。
   - 分析模块与表型或分组信息之间的相关性。
   - 输出软阈值图、聚类树、模块性状热图、模块基因信息表等结果。

5. **Venn 图**
   - 支持多个基因列表之间的交集分析。
   - 可生成 Venn 图、交集基因列表、成员矩阵和集合统计表。
   - 适合差异基因、WGCNA 模块基因和机器学习筛选基因的交集整理。

6. **富集分析**
   - 支持 GO 和 KEGG 富集分析。
   - 可生成条形图、气泡图、圈图和富集结果表。
   - 支持可选表达矩阵，用于导出输入基因对应的表达数据。

7. **机器学习**
   - 支持样本类型矩阵整理、特征值提取和多算法建模。
   - 包含常见机器学习模型，用于筛选候选特征基因并比较模型表现。
   - 输出 ROC 图、模型性能结果、Top 基因表和相关可视化文件。

8. **列线图**
   - 基于特征基因构建预测模型。
   - 输出列线图、校准曲线、决策曲线、模型 ROC 曲线、模型系数和预测概率。
   - 适合对候选基因进行风险预测模型展示。

9. **ROC 曲线**
   - 对指定基因进行 ROC 分析。
   - 输出单基因或多基因的箱线图、合并 ROC 曲线、AUC 统计图和结果表。
   - 可用于评估候选基因区分不同样本组的能力。

10. **免疫浸润**
    - 支持免疫细胞比例估计、组间比较和相关性分析。
    - 可生成免疫浸润热图、箱线图、免疫细胞相关性图、基因-免疫细胞相关性图和结果汇总表。
    - 支持后续关键基因森林图、免疫可视化和棒棒糖图分析流程。

## 输入文件要求

### 表达矩阵

表达矩阵通常为 `.txt`、`.tsv` 或 `.csv` 格式，推荐使用制表符分隔文本。

基本格式：

```text
geneSymbol    Sample1    Sample2    Sample3
GENE1         1.23       2.31       1.98
GENE2         0.52       0.73       0.66
```

要求：

- 第一列为基因名，推荐列名为 `geneSymbol`。
- 后续列为样本表达值。
- 表达值应为数值型。
- 样本名应与分组文件中的样本名一致。

### 分组文件

差异分析、WGCNA 和部分建模模块需要对照组与实验组样本列表。

示例：

```text
Sample1
Sample2
Sample3
```

### 基因列表

富集分析、Venn 图、ROC、列线图和机器学习相关模块可使用基因列表文件。

示例：

```text
GENE1
GENE2
GENE3
```

## 运行方式

### 方式一：Windows 批处理启动

在项目根目录双击以下任一文件：

```text
start_app.bat
启动软件.bat
```

### 方式二：R 命令启动

在项目根目录打开 R 或 RStudio，运行：

```r
shiny::runApp(".")
```

### 方式三：启动脚本

也可以使用项目中的启动脚本：

```r
source("scripts/launch_app.R", encoding = "UTF-8")
```

## 输出结果

不同模块会生成不同类型的结果文件，包括：

- 表达矩阵：`geneMatrix.txt`、`geneMatrix.csv`
- 差异分析结果：差异基因表、上调基因列表、下调基因列表

- 模型结果：机器学习模型性能表、Top 基因表、列线图模型结果
- 免疫分析结果：免疫细胞比例矩阵、相关性结果、组间比较表
